Компания «СТИ» основана в 1999 г. и на сегодняшний день является ведущим системным интегратором с огромным опытом реализации ИТ-проектов различного масштаба и уровня сложности. Компания разрабатывает и внедряет специализированные ИТ-решения для различных отраслей, в числе которых государственные учреждения, топливно-энергетический комплекс, финансовый и промышленный сектор.

Импортозамещение СУБД

Продукты Arenadata могут эффективно замещать аналитические и транзакционные СУБД крупнейших западных производителей:

  • Импортозамещение Teradata, Oracle, Vertica, Tanzu/Pivotal Greenplum, Cloudera и др. для хранения, аналитической и транзакционной обработки данных
  • Импортозамещение Confluent Kafka, IBM Streams, SAS Event Streams Processing, Azure Stream Analytics и др. для загрузки и потоковой обработки данных
  • Импортозамещение Microsoft SQL, Amazon Timestream, HCL Informix, InfluxDB Enterprise и др. для обработки витрин данных
  • Импортозамещение Oracle DB, IBM DB2, SAP Sybase ASE и др. для работы с данными при нагрузке смешанного типа, в первую очередь OLTP (российский аналог PostgreSQL)
  • Импортозамещение Elastic, Splunk для анализа и полнотекстового поиска в режиме реального времени

Arenadata Enterprise Data Platform (EDP)

Хранение и обработка данных

Arenadata DB (ADB)

Аналитическая, распределённая СУБД, построенная на MPP-системе с открытым исходным кодом Greenplum. Предназначена для хранения и обработки больших объёмов информации — до десятков петабайт
Arenadata Hadoop (ADH)

Аналитическая, распределённая СУБД, построенная на MPP-системе с открытым исходным кодом Greenplum. Предназначена для хранения и обработки больших объёмов информации — до десятков петабайт
Arenadata QuickMarts (ADQM)

Кластерная колоночная система управления базами данных, созданная на основе ClickHouse. Предназначена для аналитические отчёты разного плана с использованием Big Data.
Arenadata Postgres (ADPG)

Универсальная база данных для эффективной работы с нагрузкой разного профиля, в первую очередь, OLTP (Online Transaction Processing). Arenadata Postgres позволяет работать с различными объёмами данных и поддерживает широкий набор типов данных.
